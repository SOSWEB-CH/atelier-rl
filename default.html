Cette page est réservée à la page d'accueil de votre site Internet.
 
A tout moment vous avez la possibilité de mettre à jour votre site Internet en envoyant vos fichiers sur le serveur Web grâce à votre programme FTP. Si vous ne possédez pas encore d'application FTP, consulter notre page de liens utiles. Plusieurs programmes gratuits vous y seront proposés.

Lors de la connection FTP vous vous trouvez directement dans le répertoire web. C'est dans ce répertoire que vous devez envoyer les fichiers constituant votre site. Le nom de la page d'accueil de votre site doit être index.html. Elle est directement accessible par l'URL: http://www.atelier-rl.ch .


  Administrer vos adresses e-mail!
  Les administrateurs du site Internet utilisent l'URL: http://courrier.net4all.ch.
Après avoir rentré cet URL, l'administrateur doit s'identifier grâce au nom d'utilisateur et au mot de passe que lui a fourni net4all. Il est alors en mesure de gérer les comptes emails du site Internet.
Les emails peuvent être consultés à l'adresse http://www.atelier-rl.ch/webmail
